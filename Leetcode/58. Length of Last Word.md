# 58. Length of Last Word

### Problem
Given a string s consisting of words and spaces, return the length of the last word in the string.

A word is a maximal substring consisting of non-space characters only.

```
Example 1:
Input: s = "   fly me   to   the moon  "
Output: 4
Explanation: The last word is "moon" with length 4.
```
### Note
```
String fuction: strlen(s), value do not include "\0"
In C/C++, "\0" is the end of a string, so "Hello" requires 6 bytes.
```
### Solution 1
```c =
int lengthOfLastWord(char* s) {

   int i=0, temp=0, ans;

    while (s[i]!='\0'){
        if (s[i]!=' '){
            temp++; // To ensure there are no special cases, such as 'as  '
            ans= temp; 
        }
        else{
            temp = 0;
        }
        i++;
    }

    return ans;
}
```
### Issue 1 & Fix 1
```
Issue: If the string length is known, the runtime can be reduced.
Fix: Use strlen(s)
```
### Solution 2
```c =
int lengthOfLastWord(char* s) {

   int ans=0;

    for(int i=strlen(s)-1; i>=0; i--){
        if(s[i]!=' '){
            ans++;
        }
        if(ans!=0&&s[i]==' '){
            break;
        }
    }

    return ans;
}
```
