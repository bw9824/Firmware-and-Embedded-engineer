# 136. Single Number

### Problem
Given a non-empty array of integers nums, every element appears twice except for one. Find that single one.

You must implement a solution with a linear runtime complexity and use only constant extra space.
```
Example 1:
Input: nums = [4,1,2,1,2]

Output: 4
```
### Note
```
(Hint) XOR
AND: It is true only when both are true.
OR:  It is true only if either one is true.
XOR: It is true only when one is true.
=> "A number appearing twice" will always be 0 after XOR. Then we can get the single one.
```
### Solution 1
```c =
int singleNumber(int* nums, int numsSize) {
    
    int ans=0;
    for (int i=0; i<numsSize; i++){
        ans = ans^nums[i];
    }
    return ans;
}
```
