# 67. Add Binary

### Problem
Given two binary strings a and b, return their sum as a binary string.

```
Example 1:
Input: a = "1010", b = "1011"
Output: "10101"
```

### Solution 1
```c =
char* addBinary(char* a, char* b) {
    int len_a=strlen(a), len_b=strlen(b), len_ans, temp=0; // A flag to remember if it carries to the next bit

    //Step 1: We need to initialize the return char.
    if (len_a < len_b){
        len_ans=len_b+1;
    }
    else{
        len_ans=len_a+1;
    }

    char* ans = (char *)malloc(len_ans + 1);
    ans[len_ans] = '\0'; // Without '\0', printf will keep reading until it reaches garbage values.

    //Step 2: We start from the last index of char a and char b, respectively.
    //temp is to store whether there is a carry over from the previous loop.
    while((len_a>0) || (len_b>0) || (temp == 1)){
        if (len_a>0){
            temp += a[--len_a] - '0'; //- '0' to a number
        }
        if (len_b>0){
            temp += b[--len_b] - '0';
        }

        ans[--len_ans] = temp%2 + '0'; //+ '0' to a character
        temp = temp/2; 
    }

    return ans + len_ans; // Move the pointer forward, skipping unused ans index. Example "10" + "1" = "_11"
}
```
