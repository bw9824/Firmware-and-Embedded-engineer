# 28. Find the Index of the First Occurrence in a String

### Problem
Given two strings needle and haystack, return the index of the first occurrence of needle in haystack, or -1 if needle is not part of haystack.

```
Example 1:
Input: haystack = "sadbutsad", needle = "sad"
Output: 0
Explanation: "sad" occurs at index 0 and 6.
The first occurrence is at index 0, so we return 0.
```
### Note
```
If we use a for loop, we will miss this type of case.
Input: haystack = "mississippi", needle = "issip", We need to return to index haystack[2] to check again
```
### Solution 1
```c =
int strStr(char* haystack, char* needle) {
    int haystack_len = strlen(haystack);
    int needle_len = strlen(needle)-1;
    int haystack_idx=0, needle_idx=0, ans=-1;

    while(haystack_idx<haystack_len){
        if (haystack[haystack_idx]==needle[needle_idx]){
            if (needle_idx==needle_len){
                return haystack_idx-needle_idx;
            }
            needle_idx++;
        }
        else{
            haystack_idx=haystack_idx-needle_idx;
            needle_idx=0;
        }
        haystack_idx++;
    }
    return ans;
}
```
### Issue 1 & Fix 1
```
Issue 1: strlen(needle) - 1 will cause an error if the string is empty.
Issue 2: The time complexity is O(n×m), and it is very slow in the worst case.
```
